# Open Asteroids

## Objetivo

O jogador controla a nave e deve atirar nos asteroids antes que eles te atinjam.

## Controles

Use o mouse para mirar e aperte o botão esquerdo ou a tecla de espaço para atirar

## Pontuação

Cada asteroid atingido por um tiro lhe dará 1 ponto.
Sua pontuação será mostrada ao final da partida.

## Compilação e execução
Para compilar é necessário usar o visual studio 2019 e abrir a solução OpenAsteroids.sln. Use como projeto de inicialização o projeto 'OpenAsteroids'.

## Observações
Em caso de comportamentos estranhos (projétil indo na direção contrária ou asteroids não aparecendo) diminua o valor da macro 'FPS' encontrada na 'main.cpp' (por exemplo, diminuir de 60 para 30).
